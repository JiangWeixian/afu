display or parse time in effect way

## usage

```
npm install -g @ifu/daydayup
```

## features

- **translate duration**
  - `day d 360000`
  - `dai 1d` - [support shortcuts](https://day.js.org/docs/en/durations/creating#list-of-all-available-units)
- **parse time**
  - `day 1599899523680`
